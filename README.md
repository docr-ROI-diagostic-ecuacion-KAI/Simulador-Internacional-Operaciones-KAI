# Simulador Procesos Internacionales Operaciones KAI

Aplicacion web estatica para practicar operaciones internacionales en un entorno 100% simulado.

## Practicas incluidas

- Radar de mercados internacionales.
- Landed cost + ROI exportador.
- Ruta logistica inteligente.
- Agente integrador con fuentes externas simuladas.

## Front DOC ROI

La interfaz sigue el sistema DOC ROI: Inter, hero oscuro, negro/azul DOC ROI, azul hielo, pills en mayusculas, cards grandes, botones tipo pastilla y una UX de diagnostico guiado.

## Despliegue

Proyecto preparado para Vercel como sitio estatico. La entrada principal es `index.html`.
